Installation (For Developers)

Clone the repository:
Bash
git clone https://github.com/SPIDY-alt/MedCareApplication.git
cd MedCareApplication
Install dependencies:

Bash
npm install
Set up Environment Variables: Create a .env file in the root directory and add your Supabase and Firebase credentials, as well as the API endpoint for prescription extraction.

Run the application locally:

Bash
npm run dev
Build the Android App (via Capacitor):

Bash
npm run build
npx cap add android
npx cap open android
